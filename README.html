Instructions

Refer to the Walkthrough and debugging sections for details regarding Single Linked Lists. The same rules apply here.

For this Challenge you will be writing a class named LinkedCharList.java, it will be using the CharNode.java class provided to create a single linked list that represents a String. The behavior of this class will be largely like the behavior of the String class which you are already familiar with. 

Starter Code

An interface called LinkedCharListInterface.java and a Node class called CharNode.java have been provided. They are required to complete the assignment and should not be modified. They will be reset to their original contents before testing.  LinkedCharList.java must implement the given interface.

Instructions

Your LinkedCharList.java class should implement the provided LinkedCharListInterface.java interface. As in the Walkthrough and Debugging sections, all Method prototypes are provided with detailed descriptions in the interface.

Test and Submit

Tests

There is no provided RunLocalTest.java for this assignment, you should consider writing your own. The Run button in vocareum will still check style, and attempt to execute any RunLocalTest.java file that you may have created. 

Submit

Once you have completed your implementation submit the assignment. You have 10 submissions to receive the maximum points.
